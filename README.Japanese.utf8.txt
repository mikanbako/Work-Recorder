労働時間を記録・集計するためのスクリプト群

1. 使い方
  以下の例は、bash上で実行したものです。

  1. 本スクリプト群を新しく作成したディレクトリに入れます。このディレクトリ
    に、労働時間が記録されていきます。

  2. 労働時間を記録するファイルを作成します。

    以下を実行してください。

      ./create_work_record.py

  3. 労働時間を記録します。

    例えば、「example_project」というプロジェクトのために、2011/05/29の
    9:00〜12:00、13:00〜18:00の間、働いたとします。その労働時間を
    記録するには、以下を実行します。

      ./record_work_time.py example_project 20110529 900 1200 1300 1800

    労働日の形式は、YYYYMMDDもしくはMMDD、MDDです。Yは、年を表す1文字です。
    Mは、月を表す1文字です。Dは日を表す1文字です。年を省略した場合は、
    今年を表します。

    労働時刻の形式は、HHMMもしくはHMMです。Hは、時(hour)を表す1文字です。Mは、
    分を表す1文字です。労働時刻は、2つを1組として指定してください。1組の
    労働時刻が、その間の労働時間を表します。複数の組の労働時間を
    指定することもできます。

    詳しくは、以下を実行してください。
    
      ./record_work_time.py -h

  4. 労働時間を集計します。

    以下を実行すると、2011/05の労働時間をプロジェクトごとに表示します。
    複数の月にまたがった期間を指定した場合、月ごとかつプロジェクトごとに
    労働時間を表示します。

      ./aggregate_work_time.py 20110501 20110531

    期間の形式は、YYYYMMDDもしくはMMDD、MDDが2つの組です。Yは、年を表す
    1文字です。Mは、月を表す1文字です。Dは日を表す1文字です。年を
    省略した場合は、今年を表します。

    詳しくは、以下を実行してください。

      ./aggregate_work_time.py -h

2. 動作条件

  本スクリプト群の動作には、以下のソフトウェアのインストールが必要です。

  * Python 2.7 (http://www.python.org/)

3. 制限

  1. 本スクリプト群だけでは、記録した労働時間の修正・削除はできません。
     sqlite3を使用するか、それ関連のユーティリティツールをご使用ください。

4. 開発者向け情報

  1. 必要なソフトウェア

    * SCons (http://www.scons.org/)

  2. 全ユニットテストの実行

    以下を、本プロジェクトのトップディレクトリで実行してください。

      scons test

  3. 開発環境の準備

    以下を本プロジェクトのトップディレクトリで実行してください。
    本スクリプト用のPYTHONPATHを設定します。
 
      source environment.sh
